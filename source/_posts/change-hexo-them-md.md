# 更换 hexo 博客主题

在 https://hexo.io/themes/ 可以查到众多的 hexo 主题，我们可以选一款喜欢的主题，然后用到自己的博客上。

1. 首先选择喜欢的主题，然后到他的 github 上将其 clone 到 themes 文件夹的某个文件中

    git clone git@github.com:probberechts/hexo-theme-cactus.git thenes/cactus

2. 修改 `_config.yml`

   theme: cactus

3. 重启 hexo 即可
