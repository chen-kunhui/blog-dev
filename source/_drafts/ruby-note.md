# 动态获取对象的方法和属性等

- `obj.methods` : 获取对象的方法和属性
- `obj.public_methods` : 获取对象的非私有方法 
- `obj.class` : 获取对象的类型
- `obj.instance_of? class_name` ： 判断某一对象是否属于某一类型
